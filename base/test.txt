二.矿机页面
    1.显示矿机，以及当前签到的获得静动态收益总和,签到按钮
        1-1.静态收益 每个用户每天签到会获得100TAU，金额 币种 后台可设置 
        1-2.动态收益 用户的下级用户签到，也能获取的收益，返佣收益按之前上下级奖励返佣方式计算
    2.用户新增一个账户(积分账户)用于存放签到产生的收益(以上静态收益 动态收益)，当用户签到账户达到10000TAU,收益溢出部分不计算，超出10000记录10000
      如果在不激活矿机的情况下，将不能继续签到,激活矿机需要支付一定金额,矿机需要多次激活，每次激活的价格可以不同
      支付金额 币种 收款地址 后台可设置

        第二次激活 -> 条件:积分账户达到 20000TAU -> 激活需支付 20000TAU
        第三次激活 -> 条件:积分账户达到 30000TAU -> 激活需支付 10000TAU

      用户支付成功后可将积分账户的金额提现

五.空投限制
    1.用户卖出TAU亏钱状态，会有空投事件，后台空投前需要判断当前用户的持仓金额,因为空投的是usdt,所以需要换算一下
      持仓金额 = 用户钱包TAU余额 * TAU汇率
    2.后台管理系统添加一个配置项 空投条件持仓限制 这个值是一个百分比
    3.空投金额判断
        3-1.合约事件通知后台空投的金额
        3-2.通过空投持仓限制计算的金额 = 持仓金额 / 空投条件持仓限制
        3-3.根据以上两点取较小值

        例1:合约空投事件金额:100USDT 当前TAU价格:0.5USDT 空投条件持仓限制:20%
            持仓TAU:100TAU => 持仓金额:100TAU * 0.5USDT = 50USDT
            计算的空投金额 = 50USDT / 20% = 250USDT
            因为两点取较小值(100USDT < 250USDT)，所以空投金额100USDT
        
        例2:合约空投事件金额:100USDT 当前TAU价格:0.5USDT 空投条件持仓限制:20%
            持仓TAU:10TAU => 持仓金额:10TAU * 0.5USDT = 5USDT
            计算的空投金额 = 5USDT / 20% = 10USDT
            因为两点取较小值(10USDT < 100USDT)，所以空投金额10USDT
